# Elfsight Test Task (Rick and Morty API)

## Для запуска:
- скачать репозиторий
- установить все зависимости командой "npm install"
- запустить проект командой "npm start"
- проект запустится на localhost:3000


## Сделано
- устранена ошибка пагинации при попытке перехода на последнюю страницу
- закрытие popup при клике за его границей, блокировка скролла
- удален неиспользукмый popup context
- добавлены хуки useFilter(отвечает за добалвение search params для фильтрации) и useSpecies(получает список всевозможных species для фитрации и сохраняет их в local storage) useClickOutside(отлавливание кликов за пределами выбранного эллемента)
- испралены мелкие баги
- переписан компонент Card
- реализована фильтрация персонажей
